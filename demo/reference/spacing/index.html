Spacing / Padding

The spacing utility class gives you the option to space out content vertically (along the x axis) without having to use the grid system or creating custom CSS.

Having a group of set sizes (based on the grid system's gutter value) also maintains a consistency throughout a design.

Small
Gutter / 2
Default
Gutter × 1
Large
Gutter × 2
Larger
Gutter × 4

Spacing Examples

Small $gutter / 2

Default $gutter * 1

Large $gutter * 2

Larger $gutter * 4

Zeroing

Remove padding
pad-0
pad-0-t

Note that this class can also be used to remove padding from a data-bk-layout element (only x-axis of course).

Mixing options

Classes can be combined to create various configuations:

Apply normal padding with small bottom padding
class="pad pad-s-b"
Apply small bottom padding with large top padding
class="pad-s pad-l-t"
Just bottom padding
class="pad-b"
