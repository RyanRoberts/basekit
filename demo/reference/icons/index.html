Icons

Icon classes are a simple way to add icon graphics to strings of text.

Currently only arrow icons exist. Additional SVG icons for search, open, close, social media logos, etc will be added.

Arrows

Using the following classes we can add indicator arrows to links and other elements encouraging interaction.

Right

Displays a visual “call to action” after the link.

  • Sed posuere consectetur est at lobortis. Donec ullamcorper nulla non metus auctor fringilla.

    Read More

  • Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem neum sociis natoque.

    Read More

Usage class="arrow-r"

Left

Displays a visual “call to action” before the link.

  • Sed posuere consectetur est at lobortis. Donec ullamcorper nulla non metus auctor fringilla.

    Return

External Link

Indicate to the user that the link exits the website.

Visit our Facebook page to like us and stay up-to-date on coming events.
Usage class="arrow-exit"

Scroll Down

Indicate to the user that the link moves down the page.

Scroll to end
Usage class="arrow-down"

Scroll Up

Indicate to the user that the link moves up the page.

Go to start
Usage class="arrow-up"

Download

Indicate to the user that the link download a file.

Download this file
Usage class="arrow-dl"
