Colours

Colour values are almost always based on brand guidelines or style guides provided by a designer or client.

Primary and Secondary colour groups are often used for headings, links, button and background colours. Links and buttons may default to the primary colour with the secondary colour used on hover.

Often a tertiary colour option won't be provided, in these cases it is advised to find a appropriatly fitting grey combination. Tertiary colours are useful for active link states, buttons, and background colours.

Colours are set in /css/settings/_colors.scss

Brand Colours

Primary

  1. default
  2. --dark
  3. --light
// Sass
.thing {
  background-color: $primary;
}

<!-- html -->
<div class="bg-primary">…</div>
<h3 class="color-primary">…</h3>
