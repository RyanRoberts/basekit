Links

Similar to the Basekit layout attribute, links in Basekit are also applied using a data attribute: data-bk-links.

This scoping to a data attribute allows simpler configuration and less stuff in the class attribute.

Links styles

By default links are available in two main styles; primary and secondary. There are then various “modifiers” that can be applied to fine-tune link styles to suit their context.

Undefined

This link has no styles set which makes it fairly useless but you can see there is no “default”, you have to make a choice.
Usage data-bk-links=""

Primary

This link has the primary style and on hover you’ll see the secondary colour. No transitions or underlines.
Usage data-bk-links="primary"

Secondary

This link has the secondary style and on hover you’ll see the primary colour. No transitions or underlines.
Usage data-bk-links="secondary"

Fade

This link has the fade transition between the primary and secondary state. It works for all states.
Usage data-bk-links="primary fade"

Underline

This link has an underline and the fade transition.
Usage data-bk-links="primary fade underline"

Underline Hover

This link has an underline on hover and the fade transition.
Usage data-bk-links="primary fade underline-hover"

Underline Initial

This link initially has an underline and it disappears on hover.
Usage data-bk-links="primary fade underline-initial"

Dim

This link is dimmed until you hover.
Usage data-bk-links="primary fade dim"

Dark Background

This is a white link on dark background. Applying reliable color hover-states on dark background can be quite tricky unless the colour pallette has been designed specifically for this. For example, our primary and secondary styles don't look so good, particularly for WCAG compliant contrast ratios.
Usage data-bk-links="white underline-hover fade"

Dark Background Alternative

This is a white link on dark background dimmed and with the fade transtion. I find this is the best compromise of style and clarity.
Usage data-bk-links="white dim fade underline-hover"

Inherit

This link has inherited its parent text colour. When using inherit I find similar rules that apply to links on dark backgrounds also apply here.
Usage data-bk-links="inherit dim fade underline"

Big Text

Underlines look quote nice on large words with descenders…
