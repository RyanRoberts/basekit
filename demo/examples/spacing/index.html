Padding classes can be mixed up to create various configuations, for example:

  • pad-b simply adds bottom padding
  • pad-s-t adds small top padding
  • pad pad-s-b adds normal padding then changes the bottom padding to the small size
  • pad-s pad-l-bsmall top and large bottom padding
  • And so on.
