    variables

    primary

    $primary: tomato;

    Description

    Primary Brand Colour Set your primary brand colour.

    primary--light

    $primary--light: lighten($primary, 15%);

    Description

    Light Variation: Tweak the lighten value to get the colour you want, or pull it out completely if the designer has included a light colour value.

    primary--dark

    $primary--dark: darken($primary, 20%);

    Description

    Dark Variation: Tweak the darken value to get the colour you want, or pull it out completely if the designer has included a dark colour value.

    grey--light

    $grey--light: lightgrey;

    Description

    Grey is commonly used across all website designs, to simplify matters I've included three default values which should cover most situations. Change the actual values to specific colour values if CSS keyword values don't suit.

    body

    $body: whitesmoke;

    Description

    Set the body background colour here, if you want to add an image you should configure the $body-media currently in settings/media.scss

    text

    $text: #393939;

    Description

    Set the primary (body) text colour

    heading

    $heading: $text;

    Description

    This is optional, you're better off using the color utility classes to set the colour on headings unless there is very little variation in heading colours.

    alert-text

    $alert-text: white;

    Description

    Alert classes can be used anywhere but for now they're tied to form alerts, set the colours here if you want to change them.

    variables

    mq-breakpoints

    $mq-breakpoints: (
  palm: 480px,
  hands: 600px,
  lap: 768px,
  desk: 1024px
) !default;

    Description

    Define the default breakpoint name and value. Fewer will generate smaller CSS files but less control over layout.

    Author

    • Ryan Roberts

    columns

    $columns: 12 !default;

    Description

    Define the maximum number of grid columns

    12 is the value I would recommend because you can get just about any column layout combination out of it (except odd column layouts). 24 columns is also useful but increases file size and CSS.

    Used by

    • [mixin] grid-columns
    • [mixin] grid-indent

    Author

    • Ryan Roberts

    column-width

    $column-width: 70 !default;

    Description

    Define the width of each column in pixels, without the px.

    Author

    • Ryan Roberts

    gutter

    $gutter: 30 !default;

    Description

    Define the width of each gutter space between the columns, without the px. Gutters are applied either side of a column, e.g. [gutter][column][gutter][gutter][column][gutter]

    Author

    • Ryan Roberts

    margin

    $margin: $gutter / 2 !default;

    Description

    Since gutters are actually applied around a column they need to the split in half. This variable simplifies that and is the variable commonly used.

    Author

    • Ryan Roberts

    bk-grid-name

    bk-grid-mq-to

    $bk-grid-mq-to: true;

    Description

    Enable or disable max-width breakpoint CSS. In simpler designs you may not ever utilise max-width breakpoints, set this false in that case to save users downloading unnecessary CSS.

    Example

    <div data-bk="to:lap:8  to:desk:6">

    Author

    • Ryan Roberts

    [private] bk-grid-padding-small

    $bk-grid-padding-small: $margin / 2;

    Description

    By default columns get a (half gutter) margin around them, this variable defines the smaller size margin. Typically it will be half but you can tweak as necessary.

    Used by

    [private] bk-push-columns

    $bk-push-columns: $columns - 1;

    Description

    Columns can be pushed (left or right) using padding CSS, you'll never want to push the a layout the total number of columns so here we subtract one from the number.

    Used by

    • [mixin] grid-indent

    Author

    • Ryan Roberts

    bk-push-right

    $bk-push-right: true;

    Description

    Enable/Disable push (towards the right |-->) CSS.

    Used by

    • [mixin] grid-indent

    Author

    • Ryan Roberts

    bk-push-left

    $bk-push-left: true;

    Description

    Enable/Disable push (towards the left <--|) CSS. Use cases for this are quite small since blocks naturally position over to the left. However, it can be useful if you align blocks with text-align: right and then need to push a block to the left.

    Used by

    • [mixin] grid-indent

    Author

    • Ryan Roberts
